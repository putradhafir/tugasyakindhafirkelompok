LATAR BELAKANG & PENGGUNAAN APLIKASI

Kepadatan kendaraan dan kurangnya sistem parkir terintegrasi menjadi masalah utama di area perkotaan. Smart Parking Management hadir sebagai solusi cerdas menggunakan pendekatan Object Oriented Programming (OOP) untuk memodelkan kendaraan, slot parkir, transaksi, dan petugas secara real-time. Aplikasi ini dapat diintegrasikan dengan pintu otomatis dan kamera untuk identifikasi plat nomor.

// ============================================================
// CLASS 1: KENDARAAN
// Merepresentasikan kendaraan yang akan parkir.
// Properti:
//   - platNomor : String (nomor polisi kendaraan)
//   - jenis     : String (Mobil, Motor, Truk, dll)
//   - waktuMasuk: DateTime (waktu kendaraan masuk parkir)
// Method:
//   - infoKendaraan() : menampilkan informasi kendaraan
// ============================================================
class Kendaraan {
  String platNomor;
  String jenis;
  DateTime waktuMasuk;

  // Constructor
  Kendaraan(this.platNomor, this.jenis, this.waktuMasuk);
  
  // Method untuk menampilkan info kendaraan
  void infoKendaraan() {
    print("🔹 Kendaraan: $jenis | Plat: $platNomor | Masuk: ${waktuMasuk.hour}:${waktuMasuk.minute.toString().padLeft(2, '0')}");
  }
}

// ============================================================
// CLASS 2: AREA PARKIR (BASE CLASS)
// Kelas dasar untuk zona parkir.
// Properti:
//   - zona: String (nama zona parkir, misal "A1", "B2")
// Method:
//   - infoZona() : menampilkan informasi zona
// ============================================================
class AreaParkir {
  String zona;
  
  AreaParkir(this.zona);
  
  void infoZona() {
    print("📍 Zona parkir: $zona");
  }
}

// ============================================================
// CLASS 3: SLOT PARKIR (INHERITANCE dari AreaParkir)
// Merepresentasikan slot parkir individual.
// Properti tambahan:
//   - nomorSlot    : int (nomor identifikasi slot)
//   - tersedia     : bool (status ketersediaan slot)
//   - kendaraanPlat: String? (plat kendaraan yang menempati)
// Method:
//   - tempatiSlot(plat) : mengisi slot dengan kendaraan
//   - bebaskanSlot()    : mengosongkan slot
//   - infoSlot()        : menampilkan status slot
// ============================================================
class SlotParkir extends AreaParkir {
  int nomorSlot;
  bool tersedia;
  String? kendaraanPlat;

  SlotParkir(this.nomorSlot, String zona, this.tersedia) : super(zona);

  // Method untuk menempati slot parkir
  bool tempatiSlot(String plat) {
    if (tersedia) {
      kendaraanPlat = plat;
      tersedia = false;
      print("✅ Slot #$nomorSlot (Zona $zona) ditempati oleh $plat");
      return true;
    } else {
      print("❌ Slot #$nomorSlot sudah terisi!");
      return false;
    }
  }

  // Method untuk membebaskan slot parkir
  void bebaskanSlot() {
    if (!tersedia) {
      print("🆓 Slot #$nomorSlot dibebaskan.");
      kendaraanPlat = null;
      tersedia = true;
    } else {
      print("ℹ️ Slot #$nomorSlot sudah kosong.");
    }
  }
  
  // Method untuk menampilkan status slot
  void infoSlot() {
    String status = tersedia ? "🟢 Tersedia" : "🔴 Terisi (${kendaraanPlat ?? 'Unknown'})";
    print("Slot #$nomorSlot - Zona $zona: $status");
  }
}

// ============================================================
// CLASS 4: TRANSAKSI PARKIR
// Menangani perhitungan biaya dan pencetakan struk.
// Properti:
//   - id             : String (ID transaksi unik)
//   - platKendaraan  : String (plat nomor kendaraan)
//   - durasiJam      : int (lama parkir dalam jam)
//   - tarifAwal      : double (tarif untuk 1 jam pertama)
//   - tarifPerJam    : double (tarif untuk jam berikutnya)
// Method:
//   - hitungTotal()  : menghitung total biaya parkir
//   - cetakStruk()   : mencetak struk pembayaran
// ============================================================
class TransaksiParkir {
  String id;
  String platKendaraan;
  int durasiJam;
  double tarifAwal = 4000;
  double tarifPerJam = 2000;

  TransaksiParkir(this.id, this.platKendaraan, this.durasiJam);
  
  // Menghitung total biaya berdasarkan durasi
  double hitungTotal() {
    if (durasiJam <= 1) return tarifAwal;
    return tarifAwal + (durasiJam - 1) * tarifPerJam;
  }

  // Mencetak struk parkir
  void cetakStruk() {
    double total = hitungTotal();
    print("🧾 STRUK PARKIR");
    print("═" * 25);
    print("ID Transaksi : $id");
    print("Kendaraan    : $platKendaraan");
    print("Durasi       : $durasiJam jam");
    print("Tarif Awal   : Rp${tarifAwal.toStringAsFixed(0)}");
    print("Tarif/jam    : Rp${tarifPerJam.toStringAsFixed(0)}");
    print("─" * 25);
    print("Total Bayar  : Rp${total.toStringAsFixed(0)}");
    print("═" * 25);
    print("Terima kasih! 🚗");
  }
}

// ============================================================
// CLASS 5: PETUGAS PARKIR
// Merepresentasikan petugas yang mengelola parkir.
// Properti:
//   - nama  : String (nama petugas)
//   - shift : String (shift kerja petugas)
// Method:
//   - laporan(slotTerisi, totalSlot) : membuat laporan okupansi
// ============================================================
class PetugasParkir {
  String nama;
  String shift;
  
  PetugasParkir(this.nama, this.shift);
  
  void laporan(int slotTerisi, int totalSlot) {
    double okupansi = (slotTerisi / totalSlot) * 100;
    print("📊 LAPORAN PARKIR");
    print("═" * 30);
    print("Petugas      : $nama");
    print("Shift        : $shift");
    print("Slot Terisi  : $slotTerisi / $totalSlot");
    print("Okupansi     : ${okupansi.toStringAsFixed(1)}%");
    if (okupansi > 80) {
      print("⚠️  Status: PARKIR PENUH! Rekomendasi buka area baru.");
    } else if (okupansi > 50) {
      print("🟡 Status: Cukup ramai.");
    } else {
      print("🟢 Status: Tersedia banyak slot.");
    }
    print("═" * 30);
  }
}

// FUNGSI MAIN - SIMULASI SISTEM PARKIR
// ============================================================
void main() {
  print("🏢 SMART PARKING SYSTEM - SIMULASI LENGKAP");
  print("=" * 50);
  
  // =========================================
  // 1. DEMO CLASS KENDARAAN
  // =========================================
  print("\n📌 DEMO 1: CLASS KENDARAAN");
  print("-" * 40);
  Kendaraan mobil1 = Kendaraan("B 2026 XC", "Mobil", DateTime(2026, 6, 10, 9, 30));
  mobil1.infoKendaraan();
  Kendaraan motor1 = Kendaraan("AG 1234 Z", "Motor", DateTime(2026, 6, 10, 10, 0));
  motor1.infoKendaraan();
  Kendaraan truk1 = Kendaraan("N 5678 AB", "Truk", DateTime(2026, 6, 10, 11, 15));
  truk1.infoKendaraan();
  
  // =========================================
  // 2. DEMO SLOT PARKIR + INHERITANCE
  // =========================================
  print("\n📌 DEMO 2: SLOT PARKIR (INHERITANCE)");
  print("-" * 40);
  SlotParkir slot1 = SlotParkir(1, "A1", true);
  slot1.infoZona();  // method dari parent class AreaParkir
  slot1.infoSlot();
  
  print("\n--- Proses Parkir ---");
  slot1.tempatiSlot("B 1234 AB");
  slot1.infoSlot();
  
  slot1.bebaskanSlot();
  slot1.infoSlot();
  
  print("\n--- Parkir Kendaraan Lain ---");
  slot1.tempatiSlot("D 9876 ZZ");
  slot1.infoSlot();
  
  // =========================================
  // 3. DEMO TRANSAKSI & PETUGAS
  // =========================================
  print("\n📌 DEMO 3: TRANSAKSI & PETUGAS");
  print("-" * 40);
  
  // Transaksi 1
  TransaksiParkir trans1 = TransaksiParkir("TRX-001", "B 1234 AB", 3);
  trans1.cetakStruk();
  
  print("");
  
  // Transaksi 2
  TransaksiParkir trans2 = TransaksiParkir("TRX-002", "AG 5678 Z", 1);
  trans2.cetakStruk();
  
  print("");
  
  // Petugas 1
  PetugasParkir petugas1 = PetugasParkir("Budi Santoso", "Pagi (08:00-16:00)");
  petugas1.laporan(15, 20);
  
  print("");
  
  // Petugas 2
  PetugasParkir petugas2 = PetugasParkir("Siti Rahayu", "Malam (16:00-00:00)");
  petugas2.laporan(9, 20);
  
  // =========================================
  // 4. SIMULASI LENGKAP
  // =========================================
  print("\n📌 DEMO 4: SIMULASI LENGKAP");
  print("-" * 40);
  
  // Inisialisasi Petugas
  print("\n👤 PETUGAS");
  PetugasParkir petugas = PetugasParkir("Ahmad", "Pagi");
  print("Petugas: ${petugas.nama} (Shift ${petugas.shift})");
  
  // Inisialisasi Slot Parkir
  print("\n🅿️ SLOT PARKIR");
  var slots = [
    SlotParkir(1, "A", true),
    SlotParkir(2, "A", true),
    SlotParkir(3, "B", true),
    SlotParkir(4, "B", true),
    SlotParkir(5, "C", true),
  ];
  print("Total slot tersedia: ${slots.length}");
  
  // Inisialisasi Kendaraan
  print("\n🚗 KENDARAAN");
  var kendaraanList = [
    Kendaraan("B 1234 AB", "Mobil", DateTime(2026, 6, 10, 9, 0)),
    Kendaraan("AG 5678 Z", "Motor", DateTime(2026, 6, 10, 10, 15)),
    Kendaraan("D 9876 ZZ", "Mobil", DateTime(2026, 6, 10, 11, 30)),
    Kendaraan("N 1111 AA", "Truk", DateTime(2026, 6, 10, 13, 45)),
  ];
  
  // Proses Parkir
  print("\n🔄 PROSES PARKIR");
  int slotTerisi = 0;
  var transaksiList = [];
  
  for (var i = 0; i < kendaraanList.length && i < slots.length; i++) {
    var kendaraan = kendaraanList[i];
    var slot = slots[i];
    print("\n--- Parkir Kendaraan ${i+1} ---");
    kendaraan.infoKendaraan();
    if (slot.tempatiSlot(kendaraan.platNomor)) {
      slotTerisi++;
      int durasi = (i + 1) * 2; // Durasi bervariasi
      var transaksi = TransaksiParkir(
        "TRX-${(i+1).toString().padLeft(3, '0')}", 
        kendaraan.platNomor, 
        durasi
      );
      transaksiList.add(transaksi);
    }
  }
  
  // Laporan Akhir
  print("\n📋 LAPORAN AKHIR");
  petugas.laporan(slotTerisi, slots.length);
  
  // Tampilkan Transaksi
  print("\n💰 TRANSAKSI");
  for (var trans in transaksiList) {
    trans.cetakStruk();
    print("");
  }
  
  print("=" * 50);
  print("✅ Simulasi selesai! Terima kasih.");
}

KESIMPULAN

Dengan pendekatan Object Oriented Programming (OOP), sistem Smart Parking Management ini berhasil memodelkan entitas parkir secara terstruktur dan modular. Keunggulan yang diperoleh:

  • ✅ Modularitas: Setiap class memiliki tanggung jawab yang jelas (Kendaraan, SlotParkir, Transaksi, Petugas).
  • ✅ Reusability: Class dapat digunakan kembali dan dikembangkan (contoh: inheritance pada SlotParkir).
  • ✅ Maintainability: Perubahan pada satu class tidak mempengaruhi class lain.
  • ✅ Real-world Modeling: Objek mencerminkan entitas nyata di sistem parkir.
  • ✅ Scalability: Mudah menambahkan fitur baru seperti reservasi online atau integrasi IoT.

Implementasi ini menunjukkan bahwa OOP adalah pendekatan yang tepat untuk membangun sistem manajemen parkir yang cerdas, efisien, dan mudah dikembangkan di masa depan.

Insight: Sistem ini dapat diintegrasikan dengan teknologi IoT seperti sensor ultrasonik untuk deteksi slot secara real-time, kamera untuk identifikasi plat nomor otomatis, dan aplikasi mobile untuk reservasi.
