LATAR BELAKANG & PENGGUNAAN APLIKASI

Kepadatan kendaraan dan kurangnya sistem parkir terintegrasi menjadi masalah utama di area perkotaan. Smart Parking Management hadir sebagai solusi cerdas menggunakan pendekatan Object Oriented Programming (OOP) untuk memodelkan kendaraan, slot parkir, transaksi, dan petugas secara real-time. Dengan OOP, sistem ini mudah dikembangkan, mendukung berbagai tipe kendaraan, dan memberikan transparansi ketersediaan slot.

Cara penggunaan aplikasi: Sistem ini dirancang untuk petugas parkir dan manajer. Petugas dapat memantau ketersediaan slot, menempatkan kendaraan, dan membebaskan slot. Transaksi otomatis menghitung biaya berdasarkan durasi. Manajer dapat melihat laporan okupansi per zona. Aplikasi ini dapat diintegrasikan dengan pintu otomatis dan kamera untuk identifikasi plat nomor.

Mengapa OOP? & Objek Smart Parking

Pendekatan OOP memetakan entitas dunia nyata (kendaraan, slot, transaksi) menjadi objek yang memiliki data dan perilaku masing-masing. Hal ini mengurangi duplikasi kode, mempermudah perawatan, dan sangat cocok untuk pengembangan fitur seperti reservasi online.

Class Diagram Sistem

classDiagram
    class Kendaraan {
        +String platNomor
        +String jenis
        +DateTime waktuMasuk
        +infoKendaraan()
    }
    
    class SlotParkir {
        +int nomorSlot
        +String zona
        +bool tersedia
        +String? kendaraanPlat
        +tempatiSlot(plat)
        +bebaskanSlot()
        +infoSlot()
    }
    
    class TransaksiParkir {
        +String id
        +String platKendaraan
        +int durasiJam
        +double tarifAwal
        +double tarifPerJam
        +hitungTotal()
        +cetakStruk()
    }
    
    class PetugasParkir {
        +String nama
        +String shift
        +laporan(slotTerisi, totalSlot)
    }
    
    class AreaParkir {
        +String zona
        +infoZona()
    }
    
    AreaParkir <|-- SlotParkir : Inheritance
    Kendaraan *-- TransaksiParkir : Composition
    PetugasParkir --> TransaksiParkir : Association
    SlotParkir --> Kendaraan : Association

KESIMPULAN

Dengan pendekatan Object Oriented Programming (OOP), sistem Smart Parking Management ini berhasil memodelkan entitas parkir secara terstruktur dan modular. Keunggulan yang diperoleh:

  • ✅ Modularitas: Setiap class memiliki tanggung jawab yang jelas (Kendaraan, SlotParkir, Transaksi, Petugas).
  • ✅ Reusability: Class dapat digunakan kembali dan dikembangkan (contoh: inheritance pada SlotParkir).
  • ✅ Maintainability: Perubahan pada satu class tidak mempengaruhi class lain.
  • ✅ Real-world Modeling: Objek mencerminkan entitas nyata di sistem parkir.
  • ✅ Scalability: Mudah menambahkan fitur baru seperti reservasi online atau integrasi IoT.

Implementasi ini menunjukkan bahwa OOP adalah pendekatan yang tepat untuk membangun sistem manajemen parkir yang cerdas, efisien, dan mudah dikembangkan di masa depan.

Insight: Sistem ini dapat diintegrasikan dengan teknologi IoT seperti sensor ultrasonik untuk deteksi slot secara real-time, kamera untuk identifikasi plat nomor otomatis, dan aplikasi mobile untuk reservasi.
